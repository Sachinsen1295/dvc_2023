

Commands

```
Git init # to initiate Git

dvc init # to initiate data versioning

git status 

dvc add data/data.txt # to add file to track changes with git and run this command everytime after changes

git add data/data.txt.dvc # to upload now hash file

git log 

git checkout < checkout ref number>

dvc checkout < checkout ref number>

dvc checkout # to get recent version of data


```
